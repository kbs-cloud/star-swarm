## [0.24.3-alpha.1] - 2026-06-24

# Changelog - StarSwarm

## New Features
- **Empty Game Lobby Name Support**: You can now create a game lobby without entering a name, and the server will automatically assign a sci-fi/space themed name (e.g. `Hyperion-Cruiser 519`).
- **Release Integration**: Add the centralized `npm run release` script to support automated versioning, git tagging, changelog rolling, and GitHub release publishing.
